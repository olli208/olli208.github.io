'Getal onder de 10' met JavaScript

Voor dit project heb ik dat wat ik geleerd heb bij voorgaande projecten, toegepast in combinatie met JavaScript. Dit was daarom mijn eerste keer dat ik JavaScript heb gebruikt.

Ik vond het in het begin moeilijk, JavaScript is nou eenmaal een krachtige taal is, waarmee je zoveel ermee kan doen. Het uiteindelijke project wat daaruit gekomen is, is een spel. Het eeuwenoude spel ‘getal onder de 10’, om precies te zijn.

De code bevat voornamelijk functions, met veel variables erin. Daarnaast krijg je als je het getal raadt een ‘prijs’ dat random is. Dit kiest de game uit een array van foto’s. Raadt de gebruiker het getal in één keer, dan krijgt hij/zij een super speciale prijs.

Om heel eerlijk te zijn, begreep ik na dit project gemaakt te hebben, nog steeds niet zoveel van JavaScript. Misschien was het met jQuery eenvoudiger om toch JavaScript functies toe te passen, maar het is altijd goed om de basis te leren.
