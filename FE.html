Redesign CMD intranet met media queries

Tegenwoordig voelt een website al zeer verouderd, wanneer het niet responsive is. Een responsive website is een must, wanneer het op een ander apparaat word gebruikt dan een desktop/laptop. Vandaar dat de focus bij dit project lag om het responsive maken van de intranetsite van CMD.

De nummer 1 probleem waar je tegen op loopt is: waar moet alles. Je hebt nu minder ruimte voor je content, en al vooral op mobiele formaat. Het wordt aangeraden dat wanneer je een responsive website wil maken, je “mobile first” moet ontwerpen. Dat wil zeggen, dat alles eerst moet kloppen op de mobiele formaat (layout, tekst, etc.), voordat je grotere schermen aanpakt.

Het belangrijkste wat ik heb geleerd tijdens dit vak was iets waar ik niet meer zonder kan: flexbox. Met flexbox kan ik layouts zo aanpassen zoals ik wil en eenvoudig responsive maken. Ik implementeer deze techniek nu op al mijn websites.

Nu ben ik daar wel beter in geworden en let ik erop dat dat niet gebeurt. Ik begrijp waarom men zegt dat je “mobile first” moet ontwerpen.
